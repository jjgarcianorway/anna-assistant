# Anna Configurator TUI Mockups

> **Anna v1.3 "Configurator & Profiles"**
> Beautiful, minimal, autonomous configuration interface using ratatui

---

## Design Principles

- **Calm Pastels**: Same color palette as CLI (beautiful module)
- **Three-Panel Layout**: Sidebar (navigation) + Center (content) + Right (help)
- **Persistent Navigation**: Bottom bar with keyboard shortcuts
- **Consistent Icons**: Unicode symbols with graceful ASCII fallbacks
- **Focus Indication**: Highlighted selection with subtle borders

## Implementation Notes

### Ratatui Widgets
- **Layout**: `Layout::default()` with `Constraint::Percentage` for responsive panels
- **Borders**: `Block::default().borders(Borders::ALL).border_type(BorderType::Rounded)`
- **Lists**: `List::new(items).highlight_symbol("â— ")`
- **Gauges**: `Gauge::default()` for sliders (or custom widget)
- **Paragraphs**: `Paragraph::new(text)` for help text
- **Tables**: `Table::new(rows)` for module lists
